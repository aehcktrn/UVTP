# Security Policy

## Supported Versions
We currently support and actively patch the following versions of the UVTP protocol:

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability
**Do not open a public Issue for security vulnerabilities.**

If you discover a security flaw in the UVTP protocol or its reference implementation, please report it via one of the following channels:
*   **Email**: [utvp(AT)protonmail.com] (Optional: Use PGP if available)
*   **Private Message**: Contact `Prom'` through RedReaper Revolt.

Please include:
*   A detailed description of the vulnerability.
*   Steps to reproduce (PoC).
*   Potential impact assessment.

## Our Commitment
*   We will acknowledge receipt of your report within **48 hours**.
*   We will provide an estimated timeline for a fix.
*   Once fixed, we will give you credit in the release notes (unless you prefer to remain anonymous).

## Disclosure Policy
We follow a **coordinated disclosure policy**. We ask that you do not reveal the vulnerability to the public until a patch has been released. This protects the users and integrity of the protocol.
